🛒Express Products API:-</br>
A simple RESTful API built using Node.js and Express.js to manage products. </br>
This API supports CRUD operations like fetching, adding, updating, and replacing products.</br>

🚀Features:-</br>
✅Get all products</br>
✅Get product by ID</br>
✅Get products by category</br>
✅Add new produc</br>
✅Replace entire product</br>
✅Update only stock</br>
✅Update only price</br>

🛠️ Technologies Used:-</br>
Node.js</br>
Express.js</br>
JavaScript</br>

❗Error Handling:-</br>
1)Returns 404 if student not found</br>
2)Returns 404 if branch not found</br>
3)Returns 404 if no students exist</br>
