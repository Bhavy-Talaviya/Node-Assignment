🛒Express Products API:-</br>
A simple RESTful API built using Node.js and Express.js to manage products. </br>
This API supports CRUD operations like fetching, adding, updating, and replacing products.</br>

🚀Features:-</br>
✅Return complete list of states<br>
✅Return a single state by ID<br>
✅Return the state with the highest GDP<br>
✅Add a new state record<br>
✅Replace entire state record (except id)<br>
✅Replace annualBudget value<br>
✅Replace population value entirely<br>
✅Update literacyRate only<br>
✅Update gdp only<br>
✅Allow updating any provided fields without replacing entire object<br>
✅Delete state by ID<br>
✅Delete state by name<br>
✅Delete all states where literacyRate < given value<br>

🛠️ Technologies Used:-</br>
Node.js</br>
Express.js</br>
JavaScript</br>

❗Error Handling:-</br>
1)Returns 404 if student not found</br>
2)Returns 404 if branch not found</br>
3)Returns 404 if no students exist</br>
