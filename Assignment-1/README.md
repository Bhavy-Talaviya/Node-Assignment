📚 Student Management API (Express.js):- <br>
A simple RESTful API built using Node.js and Express.js to manage student data.
This API performs operations like retrieving all students, finding the topper, calculating average CGPA, filtering by branch, and more.

🚀 Features:- <br>
✅ Get all students <br> 
✅ Get student by ID <br>
✅ Get topper (Highest CGPA) <br>
✅ Get average CGPA <br>
✅ Get total number of students <br>
✅ Get students by branch <br>

🛠️ Technologies Used:- <br>
Node.js <br>
Express.js <br>
JavaScript <br> 

❗Error Handling:-<br>
1)Returns 404 if student not found<br>
2)Returns 404 if branch not found<br>
3)Returns 404 if no students exist<br>
